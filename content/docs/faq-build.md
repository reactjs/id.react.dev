### Bagaimana cara menulis komentar di JSX? {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* komentar disini */}
  Halo, {name}!
</div>
```

```jsx
<div>
  {/* Juga bekerja untuk
  komentar multi-baris */}
  Halo, {name}!
</div>
```
